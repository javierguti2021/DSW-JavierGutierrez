    ETIQUETAS MAS USADAS

    Body Indica la parte del cueto del documento, es la mas importante para construir cualquier documento.

    heade

    Aqui podemos indicar el titulo del documento.

    div

    Permite estructurar el documento por secciones, adicionando controles como plantillas.

    a

    Permite crear un enlace a una pagina web.

    href

    Es el atributo principal para conectar mediante un enlace a otroa pagina web.

    target

    Nos permitira indicar a los atributos de href si la pagina se abrira en la misma pagina o en una nueva.

    br

    Podemos indicar al navegador que haga un salto de pagina, para organizar el texto y quede mejor organizado.

    h1...h6

    permite indicar la importancia del contenido dando negrilla y tamaño del texto.

    li

    Permite crear lista, con parametros de orden y desorden.

    ol

    Permite crear listas de manera ordenada.

    ul

    Permite crear listas en desorden.

    p

    spam

    Permite agrupar el texto en un solo parrafo.

    spam

    Permite ingresar imagenes dentro del contenido de la pagina.
